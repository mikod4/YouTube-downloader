## How to run
1. Install python
2. Run "installRequirements.bat" file to install python libraries and ffmpeg
3. Open terminal and type "[*path_to_script_folder*]/python main.py" or "[*path_to_script_folder*]/python3 main.py"

## How to use
1. First thing which you will is "*Enter download path:*". Here you have to enter desired download path if left empty (just press enter) all thigs will be downloaded to script folder
2. <a id="menu"></a>Next is menu:
    ```txt
      ===Menu===
      Current path: .
      1. Change download path
      2. Download
      3. Exit
      
      Select:
    ```
    Here you can choose to change download path or to download form YouTube. **Selection is done by pressing number shown to the left of option and pressing enter**

3. If "Download" was chosen then you will see this prompt "*Enter YouTube link:*" here you enter a link to YouTube video/audio you want to download
4. After link was given you will see this menu: 
    ``` txt
    Select download type:
    1. Video
    2. Audio
    ```
    Select desired option and wait for download

5. After download was finshed you will get feedback if it was succesfull or if it failed
6. After finished download scirpt goes back to menu shown in [How to use point 2](#menu)
